# 株式帝国 追加機能アイデア集

現在の株式帝国には、株式売買、自社経営、社長アクション、敵対的M&A、買収防衛、業務提携、共同開発、資金支援が入りました。ここからさらに面白くするなら、次は「長く遊ぶ理由」「プレイヤー同士の駆け引き」「市場全体の事件」を増やすのがよさそうです。

このドキュメントでは、追加候補を優先度つきでまとめます。

## まず入れたい機能

| 機能 | 内容 | 面白さ | 実装難度 |
|---|---|---|---|
| ランキング | 総資産、自社時価総額、M&A成功数、提携数で競う | 高 | 低 |
| 市場ニュース履歴 | 過去のイベントを一覧で見られる | 中 | 低 |
| 会社詳細ページ | 他人の会社のステータス、提携、吸収履歴を見る | 高 | 低 |
| 日次ログインミッション | 1日ごとの目標達成で報酬 | 高 | 中 |
| 初心者保護 | 設立直後の会社は一定時間買収不可 | 高 | 低 |

最優先はランキングと会社詳細ページです。理由は、今ある機能をそのまま活かせるからです。プレイヤーが「誰が強いのか」「どの会社が危ないのか」「どこと組むべきか」を見られるようになります。

## ランキング案

ランキングはゲームの滞在時間を伸ばしやすいです。特に学校や友達同士で遊ぶなら、かなり効きます。

| ランキング | 計算方法 |
|---|---|
| 総資産ランキング | 現金 + 保有株評価額 |
| 自社時価総額ランキング | 自社株価 × 仮想発行株数 |
| 成長企業ランキング | 初期株価からの上昇率 |
| 社長力ランキング | 技術力 + ブランド + 信頼 + 財務 + 注目度 |
| M&A王ランキング | 吸収済み企業数 |
| 平和外交ランキング | 提携数 + 資金支援回数 |
| 炎上ランキング | 悪材料・失敗イベント回数 |

ランキングは1つだけではなく、複数ある方がよいです。総資産で勝てない人も「外交王」「M&A王」「成長企業」で目立てます。

## 会社詳細ページ

マーケットの企業カードをクリックした時、今は株価と売買が中心です。ここに会社情報をもっと出すと、他人の会社を観察する楽しさが増えます。

表示したい内容:

| 項目 | 内容 |
|---|---|
| 社長名 | オーナーのユーザー名 |
| 会社ステータス | 技術力、ブランド、信頼、財務、注目度 |
| 社風 | 勝負師、技術者集団、堅実経営など |
| 買収防衛状態 | 防衛中か、防衛レベル |
| 提携先 | どの会社と仲が良いか |
| 吸収履歴 | どの会社を買収したか |
| 最近の経営履歴 | 決算、共同開発、支援、M&Aなど |

これがあると、敵対的M&Aの前に偵察したり、提携先を選んだりできます。

## 市場ニュース履歴

今のイベント通知は流れて消えます。これを履歴として残すと、市場が生きている感じが出ます。

| ニュース種別 | 例 |
|---|---|
| 株価イベント | 新作発表、不祥事、サイバー攻撃 |
| 自社アクション | 決算発表、新商品、品質改革 |
| M&A | 敵対的M&A成立、失敗、防衛成功 |
| 友好イベント | 業務提携、共同開発、資金支援 |
| 管理者イベント | 企業追加、警告など |

ニュース履歴は `market/news` に最大50件だけ保存する形が軽いです。

## ミッション

毎日やる理由を作るならミッションが強いです。

| ミッション | 報酬 |
|---|---|
| 株を1回買う | 5,000円 |
| 株を1回売る | 5,000円 |
| 自社アクションを1回実行 | 20,000円 |
| 他社と提携する | 50,000円 |
| 会社詳細を3社見る | 10,000円 |
| ポートフォリオ評価額を増やす | 30,000円 |

ミッションは報酬を大きくしすぎない方がよいです。目的は金策ではなく、遊び方を自然に誘導することです。

## 初心者保護

敵対的M&Aがある以上、初心者保護は入れた方が安全です。

おすすめルール:

| 保護 | 内容 |
|---|---|
| 設立直後保護 | 会社設立から24時間は買収不可 |
| 低資産保護 | 総資産が一定以下のプレイヤー企業は買収不可 |
| 連続買収制限 | 同じ買収者は1時間に1回まで |
| 防衛通知 | 買収を受けた会社に警告履歴を残す |

これがないと、強いプレイヤーが弱い会社を狩り続けるゲームになりやすいです。

## 株式の深掘り

投資ゲームとして面白くするなら、売買にも少し戦略を足せます。

| 機能 | 内容 |
|---|---|
| 指値注文 | 指定価格になったら自動売買 |
| 損切り注文 | 一定価格以下で自動売却 |
| 配当 | 一部企業が定期的に現金を配る |
| 株主優待 | 特定企業保有で小さなボーナス |
| 信用取引風 | 上級者向け。リスクあり |

最初に入れるなら、指値注文よりも配当の方がわかりやすいです。

## セクターイベント

個別企業だけでなく、業界全体に事件が起きると市場感が出ます。

| イベント | 影響 |
|---|---|
| 半導体ブーム | 半導体銘柄が上がる |
| 通信障害 | 通信銘柄が下がる |
| ゲーム新時代 | ゲーム銘柄が上がる |
| 金融不安 | 金融銘柄が荒れる |
| 円高・円安 | 国内外銘柄の動きが変わる |

セクターイベントは、プレイヤー企業にもセクターを持たせるともっと面白くなります。

## 会社セクター選択

会社設立時に業種を選べるようにします。

候補:

| セクター | 特徴 |
|---|---|
| テクノロジー | 成長しやすいが荒い |
| 金融 | 財務が伸びやすい |
| ゲーム | ヒット時の上昇が大きい |
| 食品 | 安定しやすい |
| 通信 | 下落しにくい |
| 医療 | 信頼が重要 |
| エンタメ | 注目度が伸びやすい |

セクターがあると、同業ライバル、セクターイベント、提携相性が作れます。

## 契約システム

提携をもっと面白くするなら、契約に期限と条件を入れます。

| 契約 | 内容 |
|---|---|
| 技術提供契約 | 一方の技術力をもう一方が借りる |
| 広告代理契約 | ブランドが高い会社が他社を宣伝 |
| 資本提携 | 支援額に応じて両社に長期ボーナス |
| 独占契約 | 大きな効果があるが他社と提携できない |
| 共同防衛 pact | 相手が買収されそうな時に防衛ボーナス |

期限は24時間や3日などにすると、ずっと同じ提携で固定されず、関係が動きます。

## オークション

市場に「買収対象」や「大型案件」が出て、プレイヤーが入札する機能です。

例:

| 案件 | 内容 |
|---|---|
| 有望スタートアップ買収 | 落札者の技術力アップ |
| 広告枠オークション | 落札者の注目度アップ |
| 政府プロジェクト | 成功すれば信頼と財務アップ |
| 特許オークション | 技術力アップ、同業に有利 |

これはかなりゲームっぽくなります。管理者が案件を出せるようにしても面白いです。

## 管理者向け機能

運営しやすくするための機能も必要です。

| 機能 | 内容 |
|---|---|
| 市場停止ボタン | 荒れた時に株価更新を止める |
| 企業復旧 | 買収や削除された会社を戻す |
| 不適切名一覧 | 怪しい会社名を管理者が確認 |
| 強制社名変更 | 不適切名を管理者が修正 |
| イベント発行 | 管理者が市場ニュースを流す |
| 資産リセット | テスト用に全員の状態を戻す |

特に、敵対的M&Aがあるなら「企業復旧」はあった方が安心です。

## UI改善案

機能が増えてきたので、UIの整理も必要です。

| 改善 | 内容 |
|---|---|
| 自社タブを分割 | 経営、提携、M&A、履歴に分ける |
| 企業詳細を強化 | 会社情報と株価情報を同じ画面で見る |
| ニュース欄を常設 | 右側に市場ニュースを表示 |
| 検索・絞り込み | 企業名、セクター、プレイヤー名で検索 |
| お気に入り銘柄 | よく見る会社を固定 |

今後は、機能追加だけでなく整理も同じくらい重要になります。

## 優先順位

次に実装するなら、この順番がおすすめです。

1. 会社詳細ページの強化
2. ランキング
3. 市場ニュース履歴
4. 初心者保護
5. 会社セクター選択
6. 日次ミッション
7. 契約システム
8. 管理者向け復旧機能
9. セクターイベント
10. オークション

最初の3つは、今あるデータを活かせるので費用対効果が高いです。
